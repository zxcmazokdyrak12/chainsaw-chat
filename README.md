# 🪚 Chainsaw Chat (Fullstack Real-time Messenger)

A messenger set in the "Chainsaw Man" universe, created for those who appreciate the silence of the night city and Tatsuki Fujimoto's unique aesthetic. This project combines rigorous backend logic with vivid frontend animations.

## 🛠 Tech Stack
- **Frontend:** React (Vite) + JavaScript/TypeScript
- **Styling & Animation:** Tailwind CSS + Framer Motion
- **Backend:** Node.js + Express
- **Real-time:** Socket.io
- **State Management:** React Hooks

## ✨ Key Features
- **Real-time Messaging:** Instant message exchange without page reloads.
- **Dynamic Rooms:** Private room system with invite codes (General, Devils, Hunters, etc.).
- **User Experience:** 
  - "Typing..." statuses and system notifications for join/leave events.
  - Voice messages (MediaRecorder API integration).
  - **Unique Easter Eggs:** 
    - 🪚 Using CAPS-LOCK triggers a chance for a chainsaw revving sound effect.
    - ❤️ Typing "love" or other cute keywords triggers a Pochita animation with a fountain of hearts.
- **Design:** Default Dark Mode, custom fonts, and iconography inspired by the manga.

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run development server:
   ```bash
   npm run dev
   ```
